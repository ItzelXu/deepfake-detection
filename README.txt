README 
To run the project, follow these steps: 
1. Access the Google Drive 
First, access the Google Drive via the link provided above. 
2. Preprocessing
Open and run the preprocessing.ipynb notebook. This notebook performs exploratory data analysis (EDA), data cleaning, splits the data into several frames and crops the images as needed on the video frames dataset located in the  Google Drive. 
4. Model 1: MesoInception Inspired Model 
Ensure that the path to the pretrained model in the pretrain folder is correct. Verify that the cropped_images folder is accessible in the Google Drive. Run pretrained model in model1_pretrained.ipynb. 
5. Model 2: From Scratch 
Confirm that the cropped_images folder is accessible in the Google Drive. 
Open and run the model2_fromscratch.ipynb notebook. This model works with the images  available in the cropped_images folder.
